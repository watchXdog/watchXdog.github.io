#博客配置
今天开始写博客了！！ 博客原本在学校的时候是学长带着做的，对于一些功能还有如何配置自己一点也不懂。所以乘着暑假有空把原本的博客删了从0开始配置，结果配置的时候踩了一堆坑，因此来记录一下。
博客的时候跟着视频碰到了许多报错通过百度 Google等慢慢进行解决其中最常见的错误有以下几点

  • 错误原因
  1. npm install hexo-deployer-git时候报错，提示npm ERR!Error:EPERM:operation not permitted,unlink’D:hexoblognode_modulesfseventsnode_modulesperformance-nowlibperformance-now.js’这个时候就是提示权限不够应该通过管理员模式进行安装。
  2. hexo init 时候报错 提示WARN Failed to install dependencies. Please run ‘npm install’ manually! 这个时候大概是没有在空文件夹进行 解决办法是 把Hexo文件夹中的所有文件删除重新输入hexo init这时候就能成功了
  3. 在安装ssh密钥后 通过ssh密钥进行配对发生错误 这个情况我也不知道咋解决 不过有一种方法可以通过使用http地址来进行配对 通过复制你的博客地址在站点配置文件（也就是hexo文件夹中的config）改变url为你的博客地址从而实现配对

好了这就是俺的第一篇博客先水到这里 明天再来继续水
